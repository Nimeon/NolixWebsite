Nolix Java library

Software development should be more product-oriented than sex!

Therefore, Nolix provides various features that can be useful for many Java applications. For example: consistent validations for arguments, iterable wrappers for arrays, or GUIs for desktop and web. The features of Nolix feel comfortable - not technical.

  • Simple to include, simple to use, simple to move
  • Code first, no configuration files, no magic
  • Object-oriented by heart
