Nolix Java library

Software development should be more product-oriented than sex!

Therefore, Nolix provides various features that can be useful for many Java applications. For example: A validator for all common argument types or a framework to create web applications in pure Java.

The features of Nolix feel comfortable - not technical.

  • Simple to include, simple to use, simple to move
  • Code first, no configuration files, no magic
  • Object-oriented by heart
