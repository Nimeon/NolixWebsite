Nolix

Software development should be more product-oriented than sex!

Therefore, Nolix provides various features useful for many Java applications. Example: A validator for arguments of all common types or a framework to create web applications in pure Java. The features of Nolix feel comfortable - not technical.

  • Simple to include, simple to use, simple to move
  • Code first, no configuration files, no magic
  • Object-oriented by heart
