About Nolix

Software development should be more product-oriented than sex.
Therefore, Nolix provides various features that can be useful for many Java applications.
E.G. consistent validations for arguments, iterable wrappers for arrays, or a super-easy 2D GUI.
The features of Nolix feel comfortable - not technical.

  • Simple to include, simple to use, simple to replace
  • Code first, no configuration files, no magic
  • Object-oriented by heart
