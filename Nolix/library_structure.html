Main packages



ch.nolix.common Contains classes for basic purpose like containers or validators.
ch.nolix.element Contains Element classes. Elements are reproducible. That means that Elements can be represented by String or specification files.
ch.nolix.businessapi Contains APIs.
ch.nolix.system Contains classes for general purpose.
ch.nolix.template Contains templates like GUI looks.
