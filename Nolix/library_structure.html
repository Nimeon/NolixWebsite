Main packages



ch.nolix.coreapi Contains APIs for basic purposes like interfaces for containers.
ch.nolix.core Contains classes for basic purposes like containers or validators.
ch.nolix.systemapi Contains APIs for system purposes like interfaces for servers or for database adapters.
ch.nolix.businessapi Contains APIs for business purposes like interfaces for calendars or fractals.
ch.nolix.system Contains classes for system purposes like servers or database adapters.
ch.nolix.business Contains classes for business purposes like calendars or fractals.
ch.nolix.template Contains templates, e.g. GUI looks.
