Nolix containers

Description

 Provides containers with various search methods.

Packages

 ch.nolix.core.container

Documents

 Nolix containers

Create a ReadContainer for an array

String[] namesArray;
...
ReadContainer<String> names = new ReadContainer<String>(namesArray);

Get a subset from a IContainer

IContaner<Person> persons;
...
IContainer<Person> seniors = persons.getRefSelected(p -> p.getAge() > 65);

Get values from the elements of a IContainer

IContainer<Person> persons;
...
IContainer<Integer> ages = persons.to(Person::getAge);

Get average value from the elements of a IContainer

IContainer<Person> persons;
...
double averageAge = persons.getAverageByInt(Person::getAge);
